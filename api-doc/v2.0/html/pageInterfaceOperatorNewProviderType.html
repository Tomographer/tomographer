OperatorNewProviderType Interface

This is a ‘type interface.’ See Type Interfaces for more info on what that is.

operator new( ... )
Define any required or desired implementations of operator new, as may be required for particular types (see Tomographer::Tools::NeedOwnOperatorNew)
typedef OperatorNewProviderType;
A member type, typedef-ing the provider type itself as OperatorNewProviderType .
template struct OperatorNewAllocatorType { typedef .. Type; }
A template struct member, capable of specifying which allocator type to use for std::vector's and other STL types of contained value T.

For an example, see Tomographer::Tools::EigenAlignedOperatorNewProvider.
