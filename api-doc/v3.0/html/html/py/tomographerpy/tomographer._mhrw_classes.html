class tomographer.MHRWParams¶

Bases: Boost.Python.instance

Parameters for a Metropolis-Hastings random walk.

MHRWParams(step_size, n_sweep, n_therm, n_run)¶

Construct a MHRWParams instance, initializing the read-only members step_size, n_sweep, n_therm and n_run to the values given to the constructor.

See also

See the corresponding C++ class Tomographer::MHRWParams for more information about these parameters. (The interfaced class uses the template parameters CountIntType=int and StepRealType=double.)
