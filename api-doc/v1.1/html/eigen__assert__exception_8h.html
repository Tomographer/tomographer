Detailed Description

Define tools for Eigen's eigen_assert() to throw an exception instead of assert'ing.

Note
This header must be included BEFORE any Eigen header!
If you define the preprocessor symbol TOMOGRAPHER_EIGEN_ASSERT_EXCEPTION before including this file, then the macro eigen_assert() is defined such that it throws an exception upon failure. (In this case, eigen_assert() is defined to call eigen_assert_throw_exception().)

Macro like eigen_assert(), but which throws an exception.

You can use this macro in a definition of "eigen_assert()". If the assertion condition fails, then an exception of type Tomographer::Tools::eigen_assert_exception is thrown.
