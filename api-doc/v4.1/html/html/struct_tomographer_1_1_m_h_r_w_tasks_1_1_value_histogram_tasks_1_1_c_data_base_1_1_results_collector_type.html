Helper to get the results collector type.

You may use the following syntax to get the correct results collector type to use with this cdata (with or without binning analysis and correct parameter types):

typedef MyCData::ResultsCollectorType<LoggerType>::Type ResultsCollectorType;

where MyCData is your CData class.

This class also provides the task result type for convenience:

typedef MyCData::ResultsCollectorType<LoggerType>::RunTaskResultType RunTaskResultType;
